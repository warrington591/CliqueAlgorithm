Simply compile and run so as to read in an array from an input file and 
be able to compile it. Make sure to have the input file in the same folder as the .class file in order to work.

Once the program is running the user will be prompted to enter a number. This number should reflect the maximum number clique you wish to find.

The program will display whether the clique of the size the user entered is found and details which vertices make up that clique.
